# City Explorer API

## Getting Started

1. Get an API key to [Weatherbit.io](https://www.weatherbit.io/) and set in your `.env` file `WEATHER_API_KEY=<Your API Key>`

2. Install Dependencies

  ```bash
  npm install
  ```

3. Start Server

  ```bash
  npm start
  ```

4. Run Tests

  ```bash
  npm run test:open
  ```
